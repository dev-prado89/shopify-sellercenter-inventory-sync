# Shopify – SellerCenter Inventory Sync

Example architecture of an integration tool designed to synchronize inventory between Shopify and SellerCenter using C# and REST APIs.

## Problem

Manual inventory updates between platforms caused inconsistencies and operational errors.  
The goal was to automate synchronization to keep product stock aligned across systems.

## Solution

A .NET application that retrieves inventory data from Shopify and updates SellerCenter using their REST APIs.

The system processes product catalogs, matches items using SKU/UPC, and updates stock automatically.

## Features

- REST API integration
- Product catalog processing
- Data normalization between platforms
- Automated inventory synchronization
- Error handling and logging

## Technologies

- C#
- .NET
- REST APIs
- JSON
- Automation workflows

## Note

This repository contains only a high-level example of the architecture and does not include client code or credentials.
